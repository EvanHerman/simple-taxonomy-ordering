Simple drag and drop ordering for taxonomies using term meta.

== Description ==

Quickly and easily reorder taxonomies how you need with an easy to use and intuitive drag and drop interface.

== Installation ==

* Unzip and upload contents to your plugins directory (usually wp-content/plugins/).
* Activate plugin
* Custom Taxonomies: Add `'tax_position' => true` to the taxonomy arguments
* Default Taxonomies: Use the following <a href="https://gist.github.com/EvanHerman/4e83fda88d2b210dce95" title="Gist Snippet">snippet</a> to enable sorting on default taxonomies
* Re-order taxonomies as needed
* Profit
